About this Layout-->
Yes, a grayscaled layout! ^_^ This frames layout features Chii from Chobits. I couldn't really think of a title for this layout, so it's just untitled. ^^;; I also added in an abstract art that I made--which is basically the background and around Chii. ^_~ Many thanks goes to the Anime Shrine for the original image of Chii.

How to Use It-->
index.html is basically your main/home page--it has all the frames coding; so don't really tamper with it unless you're changing the page's title. ^_~ blank.html is, as it is titled, a blank page for the solid and empty parts about this frames layout. All images contained within the .zip file are important as well--so don't delete any of them. main.html is for your main section--where all your contents that aren't a part of the menu go. ^_~ So edit that file and replace all the contents in the div layer with your own stuff. menu.html is where your menu section is in--again, replace the contents there (EXCEPT the "Credits" part) with your own menu. And if you don't happen to like some of the colors or font that I chose for this layout, you can go to chii_style.css for that.
