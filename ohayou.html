& - Layout Information - &
A tables layout featuring main character Lain of the anime Serial Experiments Lain. It's a Halloween-themed layout, no doubt. Two images were simply blended together with various effects and abstract art added in for the final touch. As for the purple color scheme...well, although it's not the typical orange and black Halloween colors, it's still quite a "spooky" color seen on Halloween occasions, don't you think?

& - Using this Layout - &
All image files (excluding the headerbg, that is) in the .zip file are a part of the layout, so don't delete any of them please. The index.html file is this "preview" page with the coding of the layout. So either copy the coding into your own files, or replace the content in the index.html file with your own content. You may edit the style.css file if you don't like some of the fonts or colors I've chosen to go with this layout.
